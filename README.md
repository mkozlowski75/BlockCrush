# BlockCrush

BlockCrush is a simple block-crushing game prototype. Click on blocks in the grid to crush them and score points. Built with Phaser for easy cross-platform deployment (web/desktop).

## How to Run

1. Install [Node.js](https://nodejs.org/) if not already installed.
2. In the project root, run:
   ```
npm install
npm start
   ```
3. Open your browser to [http://localhost:8080](http://localhost:8080) to play.

## How to Build

To create a production build:
```
npm run build
```

## Deploy to Vercel

1. Login (one time):
   ```
   npx vercel login
   ```
2. Deploy production:
   ```
   npx vercel --prod
   ```

## Project Structure
- `src/` — Game source code
- `public/` — Static assets and HTML

## Requirements
- Node.js 16+
- npm
